故障排查

装不上 / 跑不起来，按症状对号入座。还没找到的欢迎开 issue。

常见症状

401 Unauthorized / 403 Forbidden
Token 没 read:packages scope，或过期了，或你还没被加进 org。先跑 echo $GITHUB_TOKEN 确认 shell 里拿到了 token；再回 GitHub PAT 页面 检查 scope。
装上了但样式全灰，按钮没颜色
Tailwind 没扫到产物。确认 globals.css 里 @source 的相对路径能从 css 文件定位到 node_modules/@forge-ui/react/dist。monorepo 里路径可能是 ../../../node_modules/...。
Cannot find module "@forge-ui/react/styles.css"
装的是太旧的版本（0.1.0 之前）。跑 pnpm update @forge-ui/react，或检查 package.json 里版本是不是 ^0.1.0。
React peer dep 警告
项目 React 版本 < 19。升到 19 或以上。Forge 里用了 React 19 的特性，18 以下用不了。
回到详细安装 →在 GitHub 开 issue
