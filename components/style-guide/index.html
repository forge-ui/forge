ColorSwatch

单个色块，左上 label + 右下 HEX pill，纯白自动描边。

Import

import {
  ColorSwatch, ColorSection,
  TypefaceBlock, TypographyWeightSample, TypographySizeRow,
} from "@forge-ui/react";

Usage

通过 tone 控制标签字色，size 控制尺寸。

Purple 5008E61EE
1<ColorSwatch label="Purple 500" hex="#8E61EE" tone="light" />

API

AttributeTypeDefaultDescription
labelstring—色卡左上角标签。
hexstring—背景色 HEX 值，右下角 pill 会显示大写去号版本。
tone'dark' | 'light'—决定 label 字色：深色底用 light，浅色底用 dark。
size'sm' | 'md' | 'lg''sm'色卡尺寸，控制宽高、label 字号与 pill 内边距。

ColorSection

一行色系，左侧标题 + 10 档色阶，自动根据 darkLabelUntil 翻转字色。

Usage

把完整 10 档色阶传入 scale，darkLabelUntil 控制字色翻转阈值。

Purple

Purple 50F1EBFD
Purple 100D3C2F8
Purple 5008E61EE
Purple 900301862
1<ColorSection
2  title="Purple"
3  darkLabelUntil={200}
4  scale={[
5    { shade: "50", hex: "#F1EBFD" },
6    { shade: "100", hex: "#D3C2F8" },
7    { shade: "500", hex: "#8E61EE" },
8    { shade: "900", hex: "#301862" },
9  ]}
10/>

API

AttributeTypeDefaultDescription
titlestring—色系标题，同时作为每个 swatch 的 label 前缀。
scale{ shade: string; hex: string }[]—色阶数组，通常 10 档（50 / 100-900）。
darkLabelUntilnumber—shade 值 ≤ 此值的用深色文字，超过则白字。
swatchSize'sm' | 'md' | 'lg''sm'每个 swatch 的尺寸。

TypefaceBlock

字体样张，大号 Aa + 字体名展示。

Usage

Typeface
AaPlus Jakarta Sans
1<TypefaceBlock name="Plus Jakarta Sans" />

API

AttributeTypeDefaultDescription
namestring—字体名称，显示在 Aa 右侧。
samplestring'Aa'大字展示内容。

TypographyWeightSample

单字重样本，用于多字重横向对比。

Usage

Display700
1<TypographyWeightSample text="Display" weight={700} sizeClass="text-2xl leading-8" />

API

AttributeTypeDefaultDescription
textstring—展示文案。
weight400 | 500 | 600 | 700—字重档位。
sizeClassstring—Tailwind 字号 / 行高 class，如 text-2xl leading-8。

TypographySizeRow

一行字号展示，自动铺 700 / 600 / 500 / 400 四档字重。

Usage

API

AttributeTypeDefaultDescription
labelstring—行标题（如 Display XL - 32px）。
textstring—每列展示文案。
sizeClassstring—Tailwind 字号 / 行高 class。
